5. Tư Vấn Bằng AI Chatbot RAG

Mô tả: Hệ thống sử dụng Engine Python ChromaDB lấy tri thức cửa hàng để tư vấn điện thoại.
Bắt đầu
Khách nhắn tin ("iPhone 16 pin bao lâu?")
Node backend gọi API tới Python Service (rag_engine)
Vector Hóa truy vấn
Lọc thông tin từ ChromaDB (Knowledge base)
Gửi Context + Prompt tới LLM
Tạo sinh câu trả lời tự nhiên
Trả kết quả ra khung chat UI
Kết thúc
🤖 Công nghệ: Python, ChromaDB, RAG
