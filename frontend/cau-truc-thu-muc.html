Cấu Trúc Thư Mục Hệ Thống (GDDA Store)
📁 GDDA_Project
  • backend - API Server xử lý logic nghiệp vụ (Node.js)
    • config - Chứa file cấu hình (database.js, passport.js...)
    • migrations - Chứa các file cập nhật/thêm dữ liệu SQL tự động
    • routes - Định nghĩa API Endpoints (auth.js, products.js, cart.js...)
    • server.js - File entry point để khởi chạy server Express
    • package.json - Chứa thông tin modules npm của backend
  • frontend - Giao diện người dùng Web & Admin (HTML/CSS/JS)
    • components - Các mảnh UI dùng chung (header.html, ai-chatbot.js...)
    • css - Định dạng giao diện tĩnh cho các pages
    • images - Tài nguyên hình ảnh, banner, sản phẩm
    • js - Các mã JavaScript xử lý logic cục bộ giao diện
    • index.html (cùng loạt file .html khác) - Các trang giao diện chính
    • so-do-*.html - Các sơ đồ thiết kế hệ thống (hoạt động, kiến trúc, tuần tự)
    • tailwind.config.js - Cấu hình framework TailwindCSS
  • rag_service - Microservice xử lý AI Chatbot & Gợi ý (Python)
    • chroma_db - Thư mục chứa Vector Database dùng lưu trí nhớ bot
    • main.py - Server kết nối AI App (giao tiếp với Node.js qua HTTP)
    • rag_engine.py - Thuật toán RAG kết nối Groq LLM truy xuất nhúng
    • recommend_engine.py - Thuật toán AI gợi ý sản phẩm dựa trên hành vi
    • add_knowledge.py - File chạy kịch bản Train cho Chatbot
    • requirements.txt - Danh sách module thư viện Python cần thiết
  • init.sql - File script gốc dựng toàn bộ cấu trúc & data cơ sở dữ liệu MySQL
  • START_SERVER.bat - File mã Script hỗ trợ chạy Backend và Frontend/Python cùng lúc trên Windows
